Termos de Confidencialidade de Dados

Checklist Tarefas e Rotinas informa:


Introdução


Este documento descreve os Termos de Confidencialidade de Dados que se aplicam ao uso dos serviços fornecidos pelo site Checklist Tarefas e Rotinas, que se dedica à organização e gerenciamento de tarefas e rotinas. O Site se compromete a proteger a privacidade e segurança dos dados pessoais dos usuários conforme as normas da Lei Geral de Proteção de Dados (LGPD).


Coleta de Dados Pessoais


Ao utilizar o Site, podemos coletar dados pessoais que incluem, mas não se limitam a, nome, e-mail, senha e outras informações necessárias para a execução de serviços relacionados ao gerenciamento de tarefas. Esses dados são coletados de forma explícita e com o consentimento do usuário.


Uso dos Dados


Os dados pessoais coletados serão usados exclusivamente para a operação do Site, incluindo a autenticação de usuários, envio de notificações e execução de funcionalidades relacionadas à gestão de tarefas e rotinas.


Compartilhamento de Dados


Os dados pessoais dos usuários não serão compartilhados com terceiros, exceto quando exigido por legislação ou quando necessário para a execução de nossos serviços, como para a hospedagem de dados em servidores de terceiros. Nesse caso, tomaremos as medidas adequadas para garantir a segurança dos dados.


Segurança dos Dados


O Site adota medidas de segurança para proteger os dados pessoais dos usuários, incluindo monitoramento de dados e controles de acesso adequados. No entanto, é importante lembrar que nenhum método de transmissão ou armazenamento eletrônico é 100% seguro.


Direitos do Usuário


O usuário tem o direito de acessar, corrigir ou excluir seus dados pessoais armazenados pelo Site, conforme estabelecido pela Lei Geral de Proteção de Dados (LGPD). Para isso, o usuário deve entrar em contato com nossa equipe de suporte.


Alterações nos Termos


O Site pode atualizar seus Termos de Confidencialidade de Dados periodicamente. Caso haja alterações, elas serão publicadas nesta página, com a data de atualização informada no topo.
